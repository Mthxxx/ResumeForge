<h2>📌 Sobre o projeto</h2>

<p>
ResumeForge é um SaaS onde usuários criam currículos profissionais com a ajuda de IA. A plataforma guia o usuário durante todo o processo de criação, sugere melhorias no texto e oferece simulações de entrevista para aumentar as chances de aprovação em processos seletivos.
</p>

<p>
O modelo de negócio funciona por <strong>crédito</strong> (pagamento por currículo gerado) e por <strong>plano</strong> (assinatura com funcionalidades avançadas).
</p>

---

<h2>✨ Funcionalidades</h2>

<p>✅ Criação de currículos assistida por IA</p>
<p>✅ Sugestões inteligentes de texto por seção</p>
<p>✅ Simulação de entrevistas com feedback da IA</p>
<p>✅ Acompanhamento personalizado por IA no plano premium</p>
<p>✅ Pagamento por crédito (por currículo) ou por plano (assinatura)</p>
<p>✅ API de pagamentos integrada</p>
<p>⚙️ Exportação do currículo em PDF <em>(em desenvolvimento)</em></p>

<h2>🚀 Como rodar o projeto</h2>

<p><strong>Pré-requisitos:</strong> Node.js 18+ e npm ou yarn instalados.</p>

<p>Clone o repositório:</p>

```bash
git clone https://github.com/Mthxxx/ResumeForge.git
```

<p>Entre na pasta e instale as dependências:</p>

```bash
cd ResumeForge
npm install
```

<p>Rode o servidor de desenvolvimento:</p>

```bash
npm run dev
```

<p>Acesse <strong>http://localhost:5173</strong> no navegador.</p>

---

<h2>💡 Modelo de negócio</h2>

<p><strong>Por crédito:</strong> o usuário compra créditos e gasta um por currículo gerado. Ideal para quem usa esporadicamente.</p>

<p><strong>Por plano (assinatura):</strong> acesso ilimitado à geração de currículos, além de funcionalidades exclusivas como simulação de entrevistas com IA e acompanhamento personalizado.</p>
